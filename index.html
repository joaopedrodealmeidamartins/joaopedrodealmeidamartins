>_ João Pedro de Almeida Martins

Desenvolvedor de Software | Java & Spring Boot | Automação & IoT

Desenvolvedor com trajetória sólida nas áreas de engenharia de software, eletrônica e automação industrial. Atualmente atua como Programador Java Pleno, desenvolvendo e mantendo aplicações corporativas com Java EE, Spring Boot, Hibernate, PostgreSQL, Angular e arquitetura de microserviços em ambiente Dockerizado.

Formado em Automação Industrial e Técnico em Mecatrônica, e cursando Engenharia de Software pela Universidade Estadual de Maringá, construiu uma base técnica sólida que une o universo do hardware ao do desenvolvimento de software. Essa combinação proporciona uma visão sistêmica e analítica, permitindo compreender desde o funcionamento físico dos equipamentos até a camada lógica das aplicações.

Possui experiência abrangente em desenvolvimento de software, atuando tanto no back-end quanto no front-end. Domina o uso de Spring Boot e Java EE na criação de APIs robustas e escaláveis, e tem sólida prática com Angular no desenvolvimento de interfaces dinâmicas e responsivas. Também desenvolve aplicações mobile (em Java e Swift) e projetos integrados a dispositivos IoT, unindo performance, usabilidade e inovação tecnológica.
